# Primetrade Frontend

This is a React-based frontend built using Vite.

It connects to the FastAPI backend and provides:

- User Registration
- User Login
- JWT-based authentication
- Protected Dashboard
- Task CRUD operations
- Automatic token persistence
- Error handling

## Tech Stack

- React (Vite)
- Axios
- React Router

## Running the Frontend

- npm install
- npm run dev


Backend must be running at:

http://127.0.0.1:8000
